REST API
a rest api for the leaderboard. just to check how they perform.
Complete API calls list

For all the API calls, you have to append /api to the URL and then use these.

Method Call Description
GET 
/all
 Returns a JSON array of all the students.
GET 
/stud/:rank
 Returns a student of a particular rank.
POST 
/stud/
 Adds a new student. Should be in the same structure that you get from the GET call.
DELETE 
/stud/:rank
 Removes a particular student from the list.
